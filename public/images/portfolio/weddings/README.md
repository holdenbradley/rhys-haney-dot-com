# Weddings

Drop wedding photos into the matching subfolder. Your best 3–5 go in `_featured/`.

- `_featured/` — top picks (auto-promoted to homepage)
- `ceremony/` — vows, processional, recessional
- `reception/` — first dance, toasts, party
- `couples/` — bride + groom portraits
- `details/` — rings, dress, decor, flowers
