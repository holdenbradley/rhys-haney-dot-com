# Photo Naming Convention

## Quick Rule
**Lowercase + hyphens + number at the end**

```
wedding-smith-ceremony-001.jpg
family-johnson-outdoor-002.jpg
headshot-sarah-corporate-001.jpg
```

## Step-by-Step

### 1. Choose Your Category
Pick one from these folders:
- `weddings/`
- `action-sports/`
- `families/`
- `graduations/`
- `headshots/`
- `sports/`
- `events/`

### 2. Name Your File

**Format:** `category-subject-type-NUMBER.jpg`

**Rules:**
- All **lowercase**
- Use **hyphens** (dashes) between words, NO SPACES
- End with a **3-digit number**: `001, 002, 003...`
- File type: `.jpg`, `.png`, `.webp`, or `.avif`

### 3. Examples

```
wedding-smith-ceremony-001.jpg
wedding-smith-ceremony-002.jpg
wedding-smith-ceremony-003.jpg

family-johnson-lifestyle-001.jpg
family-johnson-lifestyle-002.jpg

headshot-sarah-corporate-001.jpg
headshot-sara-individual-001.jpg

action-rock-climbing-001.jpg
action-rock-climbing-002.jpg

sports-football-team-001.jpg
```

## Why This Matters

The website **automatically converts your filename into caption text**. 

**File name → Auto Caption**
```
wedding-smith-ceremony-001.jpg  →  "Wedding Smith Ceremony"
family-johnson-lifestyle-001.jpg  →  "Family Johnson Lifestyle"
action-rock-climbing-001.jpg  →  "Action Rock Climbing"
```

No spaces, no caps, no numbers in the caption — the system handles it.

## Where to Upload

Each category has subfolders. Examples:

**Weddings:** `weddings/ceremony/`, `weddings/couples/`, `weddings/reception/`, `weddings/details/`

**Headshots:** `headshots/corporate/`, `headshots/individual/`, `headshots/family/`

**Action Sports:** `action-sports/rock-climbing/`, `action-sports/skiing/`, etc.

**Featured Photos:** Put your best shot in the `_featured/` folder inside each category to pin it to the top of the gallery.

---

## Checklist Before Upload

- [ ] File is lowercase? (`wedding-` not `Wedding-`)
- [ ] Words separated by hyphens? (no spaces)
- [ ] Ends with a number? (`-001.jpg` or higher)
- [ ] In the right folder? (check category + subfolder)
- [ ] Correct format? (`.jpg`, `.png`, `.webp`, `.avif`)

✅ Done! Upload to Google Drive and Holden will sync it to the live site.
